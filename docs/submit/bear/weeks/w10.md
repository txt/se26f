## Tests T29–T31 (final)

    T29 "*" wildcard fires on unknown event
    T30 absent "*" → stay-put, trace records "[?drought] s->s"
    T31 500 random seeds: no crash, conservation holds, no deadlock

```lua
-- T29
local bear = make_bear()
bear.states.sleep.transitions["*"] = "flee"
bear.queue = {"earthquake"}
M.start(bear)
assert(bear.current == "flee")

-- T30
local bear2 = make_bear()  -- no "*" in sleep transitions
bear2.queue = {"earthquake"}
M.start(bear2)
assert(bear2.current == "sleep")
local found = false
for _,line in ipairs(bear2.trace or {}) do
  if line:find("%[%?earthquake%]") then found=true end end
assert(found)

-- T31: chaos
local failures = {}
for seed=1,500 do
  math.randomseed(seed)
  local N = math.random(1,20); local M_n = math.random(1,8)
  local bears={};  local plants={}
  for i=1,M_n do bears[i]  = make_bear()  end
  for i=1,N   do plants[i] = make_plant() end
  local agents={}
  for _,b in ipairs(bears)  do agents[#agents+1]=b end
  for _,p in ipairs(plants) do agents[#agents+1]=p end
  local ok,err = pcall(function()
    for tick=1,30 do
      for _,a in ipairs(M.shuffle({table.unpack(agents)})) do
        M.step(a) end end end)
  if not ok then
    failures[#failures+1]=string.format("seed=%d: %s",seed,err) end
end
assert(#failures==0, table.concat(failures,"\n"))
```

## TLAs

| # | TLA | Expansion | Definition | Bear example |
|---|-----|-----------|------------|--------------|
| 28 | OA | Observability (as a design requirement) | A system is observable if you can understand its internal state from its outputs alone. Not an afterthought. | The trace log is designed in from S1. Post-run: who ate what, how many droughts, which bear starved. |
| 29 | SvD | Scalability via Data not Code | A system scales by changing configuration, not by adding branches or rewriting logic. | N=100, M=40 requires zero code changes. The generator parameters are the only variable. |
| 30 | MT | Mutation Testing | Deliberately break the code in small ways; assert your tests catch each break. Tests that miss mutations are not tests. | Break one transition target; assert at least one of the 500 seeds catches it. If none do, the tests are theatre. |

## Anti-Patterns

| # | Name | Symptom | Consequence | Bear smell |
|---|------|---------|-------------|------------|
| 28 | Log-Free System | No trace; post-mortem impossible; "it crashed but I don't know where" | Production bugs are irreproducible; debugging is archaeology | Running the full ecosystem with no trace — when Bear#3 starves, you cannot see why |
| 29 | Determinism Assumption | Tests always run agents in the same order; pass every time | Race conditions invisible until production; order-dependent bugs hide forever | `for _,b in ipairs(bears)` — bears always go first; plants always second; starvation never surfaces |
| 30 | Regression-Free Deployment | Ship without pinning any golden run; no baseline to diff against | Subtle behaviour changes invisible; "it still runs" ≠ "it still works" | 500-seed suite added in S10; no golden seeds pinned from S1 to catch silent regressions |

## Python Idioms

| # | Idiom | Bad (newbie) | Good (idiomatic) |
|---|-------|--------------|-----------------|
| 28 | `logging` module | `print(f"[{e}] {s} -> {target}")` everywhere | `logging.getLogger(f"bear.{agent.id}").info(...)` — per-agent logger; filter, level, file output; essential at N=100 |
| 29 | `heapq` for event priority | `queue.sort()` every tick — O(n log n) | `heapq.heappush(queue,(tick,event))` — O(log n); events fire in tick order across all agents |
| 30 | `hypothesis` property testing | `for seed in range(500): run(seed)` by hand | `from hypothesis import given, strategies as st; @given(st.integers())` — Hypothesis finds falsifying inputs automatically |
