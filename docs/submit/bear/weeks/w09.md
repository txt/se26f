# Week 9 — Machine Generator (N Plants, M Bears)

## The Bear

Uncomment `-- S9` blocks.

```
-- seed 42  N=5 plants  M=3 bears
-- tick 1
Plant#3 budding.   Plant#4 budding.   Plant#5 budding.
Plant#6 budding.   Plant#7 budding.
Bear#1 sniffs.  hunger=72  → Plant#6 not ripe
Bear#2 sniffs.  hunger=88  → Plant#3 not ripe
Bear#3 sniffs.  hunger=45  → sleeps

-- tick 3
Plant#6 RIPE!  yield=18
Bear#2 gets there first. hunger=88 → 58.  Plant#6 eaten.
Bear#1 blocked — Plant#6 gone. foraging on.

=== SUMMARY ===
food produced: 67   food consumed: 67   store trips: 1
conservation: OK    seed: 42
```

The ecosystem is alive. The summary is the invariant.

## Tests T28–T30

    T28 10 plants + 5 bears run to completion without error
    T29 conservation: total food consumed == total food produced
    T30 make_plant / make_bear produce lint-clean machines

```lua
-- T28
math.randomseed(42)
local bears  = {}; local plants = {}
for i=1,5  do bears[i]  = make_bear()  end
for i=1,10 do plants[i] = make_plant() end
local agents = {}
for _,b in ipairs(bears)  do agents[#agents+1]=b end
for _,p in ipairs(plants) do agents[#agents+1]=p end
local ok, err = pcall(function()
  for tick=1,50 do
    for _,a in ipairs(M.shuffle({table.unpack(agents)})) do
      M.step(a) end end end)
assert(ok, err)

-- T29: conservation invariant
local produced = 0; local consumed = 0
for _,p in ipairs(plants) do
  produced = produced + (p.total_yielded or 0) end
for _,b in ipairs(bears) do
  consumed = consumed + (b.total_eaten or 0) end
local on_plants = 0
for _,p in ipairs(plants) do
  if p.current=="ripe" then on_plants=on_plants+p.yield end end
assert(produced == consumed + on_plants,
  string.format("conservation violated: %d != %d + %d",
    produced, consumed, on_plants))

-- T30: lint clean
for _,a in ipairs(agents) do
  assert(#M.lint_collect(a) == 0,
    "lint failed for "..a.name) end
```

## TLAs

| # | TLA | Expansion | Definition | Bear example |
|---|-----|-----------|------------|--------------|
| 25 | FP | Factory Pattern | A function that creates and returns fully-configured objects. Replaces copy-paste construction. | `make_plant(kind,yield,grow_time)` generates a valid, lint-clean, id-stamped machine. No copy-paste. |
| 26 | EBI | Emergence from Bottom-up Interaction | Complex system behaviour arises from many simple agents following simple rules. | N=20 plants, M=8 bears, identical FSMs → competition, starvation, store trips. No "competition" code exists. |
| 27 | CI | Conservation Invariant (test oracle) | A quantity that must be preserved across all operations. Violations indicate a bug regardless of how they occur. | `food_produced == food_consumed + food_on_plants` holds for every seed if the system is correct. |

## Anti-Patterns

| # | Name | Symptom | Consequence | Bear smell |
|---|------|---------|-------------|------------|
| 25 | Copy-Paste Factory | Five bear machines defined by duplicating the rules table | One fix must be applied five times; one miss creates divergence | `local bear1_rules = {...}; local bear2_rules = {...}` — identical except the name |
| 26 | Hardcoded Scale | `N=5` and `M=3` burned into logic, not parameters | Cannot run chaos tests at N=100; demo is always the same | `for i=1,5 do` instead of `for i=1,N_PLANTS do` |
| 27 | N+1 Query | Each bear queries each plant individually per tick, in a loop | O(M×N) queries per tick; at N=100,M=40 this is 4000 calls | `for _,plant in ipairs(plants) do bear:query(plant) end` inside the bear's per-tick update |

## Python Idioms

| # | Idiom | Bad (newbie) | Good (idiomatic) |
|---|-------|--------------|-----------------|
| 25 | List comprehension for generation | `bears=[]; for i in range(M): bears.append(make_bear())` | `bears = [make_bear() for _ in range(M)]` — one line, same semantics |
| 26 | `zip` over parallel agent lists | `for i in range(len(bears)): pair(bears[i], plants[i])` | `for bear,plant in zip(bears,plants)` — no index arithmetic, readable |
| 27 | `enumerate` | `for i in range(len(agents)): print(i, agents[i])` | `for i,agent in enumerate(agents)` — idiomatic, no `len()`, no indexing |
