### Herdr Coding Agent

https://pyshine.com/herdr-Agent-Multiplexer-for-AI-Agents/

Overview: Herdr is an open-source, terminal-native agent multiplexer designed specifically for managing multiple coding agents side-by-side. It aims to provide a lightweight, responsive UI/TUI that works over SSH/VPS, making it suitable for various development environments.

Key Features

- Workspaces with Tiled Panes: Similar to tmux but purpose-built for agents. "Workspaces with tiled panes, like tmux but purpose-built for agents"- Automatic Agent Detection: Reads foreground processes and terminal output to determine the state of agents (working, blocked, idle, done).
- Automatic agent detection: it reads foreground process + terminal output to determine state (working, blocked, idle, done)"
- Sidebar for Triage: Rolls up each workspace to its most urgent signal, allowing for quick triage across projects. "Sidebar that rolls up each workspace to its most urgent signal so you can triage across projects"
- Socket API: Agents can use the API to create panes, spawn other agents, read output, and wait for state changes. "Socket API that agents themselves can use: create panes, spawn other agents, read output, wait for state changes"
- Session Persistence: Ensures your sessions are saved and can be restored. "Session persistence, 9 themes, mouse as first citizen, sound/toast notifications"
- Supported Agents:
  Herdr supports a variety of coding agents, including: Claude Code, Codex, pi, Droid, Amp, OpenCode, and more.
